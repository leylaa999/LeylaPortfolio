Duolog

An interactive platform for performing arts students, awarded 2nd place at ICHack.

Overview

Duolog is an online platform designed for performing arts students, enabling them to rehearse scripts with AI-driven text-to-speech capabilities. It supports multiple characters and customizable playback options.

Key Features

  • Text-to-speech API with customizable voices and speeds.
  • Supports unlimited characters for rehearsal scenarios.
  • Interactive interface enhancing script practice efficiency.
  • Collaborative rehearsal features for group performances.

Gallery

Development Process

Built during ICHack, Duolog integrates advanced APIs to provide performing arts students with an innovative rehearsal tool. The platform focuses on accessibility, usability, and enhancing the learning experience through interactive technology.
